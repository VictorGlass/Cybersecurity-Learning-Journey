# Velociraptor Endpoint Triage Lab

## Overview

This lab focuses on endpoint triage acquisition and forensic artifact collection using Velociraptor.

The objective of the lab is to practice DFIR workflows, evidence collection, and endpoint investigation techniques commonly used in SOC and Incident Response environments.

---

## Technologies Used

- Velociraptor
- Windows 10 / 11
- VMware Workstation
- Kali Linux

---

## Topics Covered

- Endpoint Security
- DFIR fundamentals
- Forensic artifact collection
- KAPE targets
- Offline collectors
- Evidence encryption

---

## Lab Objectives

- Deploy and configure Velociraptor
- Explore forensic artifacts
- Generate offline collectors
- Perform endpoint triage acquisition
- Protect collected evidence using encryption

---

## Key Concepts

### Endpoint Triage

Rapid acquisition of relevant forensic data from potentially compromised endpoints.

### Offline Collectors

Standalone executables capable of collecting forensic artifacts without requiring a persistent agent.

### Artifact Collection

Collection of:
- Event logs
- System information
- Registry artifacts
- User activity
- Security-related evidence
