# Velociraptor Triage Acquisition

## Overview

This lab focuses on endpoint triage acquisition using Velociraptor and KAPE artifacts.

The objective is to collect relevant forensic artifacts from endpoints for DFIR investigations.

---

## Key Concepts

### Triage Acquisition

Rapid collection of forensic evidence from compromised endpoints.

Purpose:
- Preserve evidence
- Accelerate investigations
- Identify malicious activity

---

## Windows.KapeFiles.Targets

Artifact used to:
- Collect forensic artifacts
- Package relevant files
- Support DFIR investigations

---

## Offline Collectors

Velociraptor supports offline collectors that:
- Execute artifacts without an active agent
- Generate standalone binaries
- Allow remote evidence acquisition

---

## Encryption Methods

### Password Encryption
ZIP protected with password.

### X509 Encryption
Uses public certificates for secure encryption.

### PGP Encryption
Uses PGP-based encryption for evidence protection.
