# Wireshark PCAP Analysis Lab

## Overview

This lab contains network traffic analysis exercises focused on identifying malicious activity using Wireshark and tshark.

Topics covered:
- Webshell activity
- Command and Control (C2)
- Reverse shells
- DNS exfiltration
- Threat hunting
- DFIR fundamentals

---

## Tools Used

- Wireshark
- tshark
- Kali Linux
- VMware Workstation Player

---

## Challenge 1

Analysis of malicious HTTP traffic involving:
- ASPX webshell upload
- Command execution
- certutil abuse
- Netcat reverse shell
