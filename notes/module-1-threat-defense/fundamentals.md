# Module 1 - Cyber Threat Defense

## Fundamentals

## 1. CIA Triad

The CIA triad is a core concept in cybersecurity:

- **Confidentiality**  
  Information should only be accessible to authorized users.

- **Integrity**  
  Data must remain accurate and unaltered.

- **Availability**  
  Systems and data must be accessible when needed.


## 2. Identification and Authentication

- **Identification**  
  Associating actions with a specific user.

- **Authentication**  
  Verifying the identity of a user.

- **Non-repudiation**  
  Ensuring that a user cannot deny their actions.


## 3. Access Control

### Principle of Least Privilege

Users should only have the minimum permissions necessary.

This reduces the attack surface and limits potential damage.


### Physical Access Control

Examples include:
- Biometrics  
- ID cards  
- Locks  
- CCTV  
- Security personnel  


### Access Control Models

#### Discretionary Access Control (DAC)
Resource owners define access permissions.

#### Mandatory Access Control (MAC)
Access is based on security levels.

#### Role-Based Access Control (RBAC)
Access is assigned based on roles.

#### Attribute-Based Access Control (ABAC)
Access depends on attributes like user, resource, and environment.
